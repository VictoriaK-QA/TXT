2. Клонировать репозиторий TXT на локальный компьютер - git clone git@github.com:VictoriaK-QA/TXT.git
3. Внутри локального TXT создать файл “new.txt” - cd TXT/       touch new.txt
4. Добавить файл под гит - git add .
5. Закоммитить файл - git commit -m "new file"
6. Отправить файл на внешний GitHub репозиторий - git push
7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT vim new.txt
Нажать i

name - Victoria
age - 28
pet - 1 
salary - 300000
Нажать Esc :wq Enter

8. Отправить изменения на внешний репозиторий - git commit -am "info about me" git push
9. Создать файл preferences.txt - touch preferences.txt
10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT
vim preferences.txt

Нажать i

favorite movie: The Godfather,
favorite sitcom: Friends,
favorite food: pasta,
favorite season: autumn,
country to travel: Italy	

Нажать Esc :wq Enter
